Probleemstelling:
Een IT bedrijf wilt een systeem waarin ze hun klanten kunnen opslaan. Voornamelijk hun NAW gegevens maar ook kritieke gegevens zoals server adres en gebruikersnaam / wachtwoord.
Deze data willen ze afgeschermd hebben van mensen die geen toegang hebben en moet dus beveiligd zijn met een gebruikersnaam en wachtwoord die voor de medewerker wordt aangemaakt.
Er moet een onderscheid komen tussen bedrijven en particulieren in het programma.
